Parte 1 - Fontes Instaladas no Computador

  1. Abra um editor de texto ou software de design (ex.: Word, Powerpoint, Canva, Iknscape).
  2. Localize a lista de fontes disponíveis no computador
  3. Escolha 5 fontes diferentes e responda:
    • Nome da fonte
    • Classificação (Serifada, Sem Serifa, Monoespaçada, Decorativa, etc.)
    • Onde você utilizaria essa fonte (ex.: site, cartaz, logotipo, apresentação)
    • Sua opinião: a fonte é fácil de ler? Por quê?

Fontes Utilizadas

  1. DejaVu Sans. Sem Serifa. Utilizaria para algum texto. Sim, é fácil de ler
  2. Bitstream Charter. Serifada. Utilizaria para expressar algo mais luxouso, como um título de uma loja de vinhos. Sim, é fácil de ler, mas pode ser mais complicada que fontes sem serifa.
  3. Bebas Neue. Display / Decorativa. Utilizaria para títulos chamativos, palavras-chaves em um banner, etc. Sim, é de fácil leitura e possui somente letras em caixa alta.

Parte 2 - Pesquisa de Novas Fontes

  1. Pesquise na internet 3 fontes para instalar (ex.: Google Fonts, DaFont, Adobe Fonts)
  2. Para cada fonte pesquisada, responda:
    • Nome da fonte
    • Site onde encontrou
    • Tipo de fonte
    • Para qual tipo de projeto ela seria indicada
    • Por que você escolheu essa fonte?
