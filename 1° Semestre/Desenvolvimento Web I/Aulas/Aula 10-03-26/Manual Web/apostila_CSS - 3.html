HTML5 Seletores CSS Box Model Flexbox

1. Recapitulação HTML5

O HTML cria a estrutura. Lembre-se das tags semânticas:

<header>, <nav>, <main>, <section>, <footer>

2. Seletores CSS

Existem três formas principais de aplicar estilo:

  • Por Tag: Altera todos os elementos daquele tipo (ex: p { ... }).
  • Por Classe (.): Altera elementos com a classe específica (ex: .botao { ... }).
  • Por ID (#): Altera UM elemento único (ex: #logo { ... }).
h1 { color: blue; }        /* Tag */
.card { padding: 20px; }   /* Classe */
#topo { height: 100px; }    /* ID */

3. O Modelo de Caixa (Box Model)

Todo elemento no HTML é visto pelo CSS como uma caixa composta por:

CONTEÚDO (Content)
(Aqui tem Padding, Border e Margin ao redor)
  • Content: Onde fica o texto/imagem.
  • Padding: Espaço interno (entre o texto e a borda).
  • Border: A linha que envolve o preenchimento.
  • Margin: Espaço externo (distância entre esta caixa e as outras).

4. Layout com Flexbox

O Flexbox é usado para alinhar elementos em linhas ou colunas facilmente.

Item 1
Item 2
Item 3
.container {
    display: flex;
    justify-content: center; /* Alinha horizontalmente */
    align-items: center;     /* Alinha verticalmente */
    gap: 10px;               /* Espaço entre itens */
}

5. Estilização de Texto

Propriedades mais comuns:

  • color: Cor da fonte.
  • font-size: Tamanho da letra (ex: 16px, 1.2rem).
  • font-weight: Espessura (bold, normal).
  • text-align: Alinhamento (center, left, justify).
