A Importância do Box Model

Um pleno entendimento do Box Model nos permite organizar os elementos da forma que quisermos.

Elementos com padrão Display: Inline

Isso signica que, por padrão, o elemento vem com display: inline.

Exemplos de elementos com padrão box:

  • <span>
  • <img src="">
  • <a href="">
  • <strong
  • <em>
  • <b>
  • <i>
  • <button>
  • <input>

Elementos com padrão Display: Block

Isso significa que, por padrão, esses elementos vem com display: block.

Exemplos de elementos com padrão box:

  • <h1> , <h2>, .., <h5>, <h6>
  • <p>
  • <div>
  • <header>
  • <main>
  • <footer>
  • <article>
  • <ul>
  • <li>
  • <section>

Comparativo dos dois tipos com exemplo Visual

Característica Block Inline
Cria nova linha? Sim Não
Largura Padrão 100% do elemento-pai Apenas o conteúdo
Aceita Width/Height? Sim Não
Alinhamento Empilhados verticalmente Lado a lado horizontalmente.

Dica Extra:

Nota: existe também o atributo display: inline-block. Que aceita atributos como width e height, alinhamentos. Mas funciona também como um atributo inline. Sua largura padrão é o conteúdo.

Ele é ótimo para criar vários botões enfileirados lado a lado.
